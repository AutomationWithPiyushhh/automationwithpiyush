SOP-01 Full Body Scan

To capture the entire viewport, we must typecast the WebDriver instance to the TakesScreenshot interface. This allows us to call the getScreenshotAs method, which returns a file object representing the screenshot.

import org.openqa.selenium.OutputType;
import org.openqa.selenium.TakesScreenshot;

TakesScreenshot tks = (TakesScreenshot) driver;
File source = tks.getScreenshotAs(OutputType.FILE);
File destination = new File("./scans/fullbody.png");
FileHandler.copy(source, destination);

SOP-02 Targeted Biopsy (Element)

Modern Selenium (v4+) allows capturing a specific organ (WebElement) without the surrounding noise. We call the method directly on the element found.

WebElement organ = driver.findElement(By.id("heatmap"));

File source = organ.getScreenshotAs(OutputType.FILE);
File dest = new File("./scans/organ_biopsy.png");
FileUtils.copyFile(source, dest);
