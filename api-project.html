Implementation Logic

Separated Routes (URLs) from Endpoints (Implementation) to maintain clean code. Implemented Chaining to pass response data (e.g., Auth Token, ID) to subsequent requests automatically.

Technical Capabilities

POJO & Lombok

Used Plain Old Java Objects (POJO) for Request/Response payloads. Leveraged Lombok annotations (@Getter, @Setter, @Builder) to eliminate boilerplate getter/setter code.

Data Driven Testing

Integrated Apache POI to fetch dynamic test data (multiple user payloads) from Excel sheets, allowing rigorous testing of endpoints with varied datasets.

Auth & Schema Validation

Robust handling of Bearer Tokens and OAuth 2.0. Implemented JsonSchemaValidator to ensure response structure compliance with contract definitions.

Extent Reports

Customized reporting listeners to log Request Body, Response Body, Status Codes, and Headers directly into the HTML report for full traceability.
