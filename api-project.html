Implementation Logic

Separated Routes from Endpoints to maintain clean abstraction. Implemented Request Chaining, dynamically extracting generated User IDs from POST responses to automatically drive subsequent GET, PUT, and DELETE validation scripts.

Technical Capabilities

POJO & Jackson

Used Plain Old Java Objects (POJO) mapped via Jackson for dynamic JSON Serialization. Leveraged Lombok to eliminate boilerplate getter/setter code, ensuring type-safe payloads.

Dynamic Data Driven Core

Integrated Apache POI to fetch datasets. Engineered dynamic string injection (timestamps) into test data at runtime to prevent 422 Unprocessable Entity errors caused by duplicate database records.

Auth & Contract Testing

Secure handling of OAuth 2.0 Bearer Tokens via externalized config files. Implemented JsonSchemaValidator to enforce strict backend contract compliance and detect structural shifts.

Extent Reports & CI/CD

Designed custom ITestListener logic to generate interactive HTML reports logging full request/response bodies and stack traces. Ready for Jenkins integration via Maven command line.
