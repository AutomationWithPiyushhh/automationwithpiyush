CRM Automation Framework

A high-performance Industrial-Grade framework built for rapid, reliable, and scalable testing of enterprise CRM modules.

Java Selenium WebDriver TestNG Maven Apache POI

Architecture

vtiger-crm-framework

├── src/main/java

│   ├── base_utility

│   ├── generic_utility

│   └── object_repo

├── src/test/java

└── pom.xml

Modular Page Object Model (POM) structure implemented for maximum maintainability and object reusability.

Framework Features

  • Hybrid Framework Architecture

    Seamlessly combines Data-Driven (Excel) and Keyword-Driven methodologies to handle dynamic test scenarios.

  • Automated Failure Analysis

    Integrated TestNG Listeners to trigger automatic screenshot capture and logging upon script failure.

  • Parallel Browser Execution

    Configured for cross-browser testing, significantly reducing total execution time for regression suites.

  • Zero Hardcoding Policy

    All environment variables and test data are strictly externalized in .properties and .xlsx files.

Advanced Reporting

Every execution generates a comprehensive Extent Report. These reports include detailed step-by-step logs, execution timestamps, and embedded failure screenshots for immediate debugging.
