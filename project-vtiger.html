Engineering Highlight

Implemented Singleton Design Pattern for property file loading and Factory Pattern for object initialization, ensuring optimal memory usage during large suite executions.

Technical Implementation Details

Hybrid Methodology

A strategic blend of Data-Driven (Apache POI for Excel) and Keyword-Driven approaches. Test data is completely isolated from script logic, allowing non-technical stakeholders to update scenarios via Excel.

Thread-Safe Execution

Utilized Java ThreadLocal<WebDriver> to ensure thread safety during Parallel Execution. This prevents session conflict when running cross-browser tests simultaneously on Selenium Grid.

Failure Analysis (Listeners)

Integrated ITestListener interface to capture screenshots automatically upon test failure. Logs are injected directly into Extent Reports for immediate root cause analysis.

CI/CD Integration

Fully Dockerized environment. The suite is triggered via Jenkins pipelines using Maven profiles (mvn test -PRegression), enabling headless execution on Linux servers.
